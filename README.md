# CMSC 126 Lab 3 – Inside UPV

A simple HTML website showcasing **University of the Philippines Visayas (UPV)**.

## Pages

- **Home**: Welcome page with featured sections.
- **Gallery**: Collection of images.
- **Videos**: Embedded videos.
- **About Us**: Information about the project and its creators.

## Project Structure

```
│
├─ index.html
├─ gallery.html
├─ videos.html
├─ about.html
├─ assets/
```

## Setup (Recommended)

1. Clone the repository:

   ```bash
   git clone https://github.com/andrianllmm/cmsc126-ctrl-celsius-lab3-html.git
   ```

2. Install **Live Server** globally:

   ```bash
   npm install -g live-server
   ```

3. Open a terminal in the project folder and run:

   ```bash
   live-server
   ```

4. Your default browser will open the site and automatically refresh on changes.
